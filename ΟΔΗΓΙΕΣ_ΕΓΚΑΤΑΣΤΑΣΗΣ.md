# Οδηγίες Εγκατάστασης — Σύστημα Κρατήσεων

## Τι περιέχει
- `index.html` — Δημόσια σελίδα κρατήσεων (για ασκούμενους)
- `admin.html` — Panel διαχείρισης (μόνο για σένα)

---

## Βήμα 1: Δημιουργία Firebase project (δωρεάν)

1. Πήγαινε στο **https://console.firebase.google.com**
2. Κλίκ **"Create a project"** → δώσε ένα όνομα (π.χ. `gym-bookings`)
3. Απενεργοποίησε το Google Analytics (δεν χρειάζεται) → **Create project**

---

## Βήμα 2: Ενεργοποίηση Firestore Database

1. Αριστερό μενού → **Build → Firestore Database**
2. Κλίκ **"Create database"**
3. Επίλεξε **"Start in test mode"** (θα το ασφαλίσουμε παρακάτω)
4. Επίλεξε region: **europe-west** → **Done**

---

## Βήμα 3: Ενεργοποίηση Authentication

1. Αριστερό μενού → **Build → Authentication**
2. Κλίκ **"Get started"**
3. Επίλεξε **Email/Password** → ενεργοποίησε → **Save**
4. Κλίκ **"Add user"** → βάλε το email σου και έναν κωδικό → **Add user**
   - Αυτά θα είναι τα στοιχεία σύνδεσής σου στο admin panel

---

## Βήμα 4: Κανόνες ασφαλείας Firestore

Αριστερό μενού → **Firestore → Rules** → αντέγραψε αυτό:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /slots/{slotId} {
      // Ο καθένας βλέπει διαθέσιμα slots
      allow read: if true;
      // Μόνο authenticated (εσύ) μπορεί να δημιουργεί/διαγράφει
      allow create, delete: if request.auth != null;
      // Κράτηση: επιτρέπεται αν το slot ήταν available
      allow update: if request.auth != null ||
                       (resource.data.available == true && request.resource.data.available == false);
    }
  }
}
```

Κλίκ **Publish**.

---

## Βήμα 5: Αντιγραφή Firebase config

1. Αριστερό μενού → ⚙️ **Project Settings**
2. Κάτω στο "Your apps" → κλίκ **</>** (Web app)
3. Δώσε ένα nickname (π.χ. `gym-web`) → **Register app**
4. Αντέγραψε το `firebaseConfig` object που εμφανίζεται

---

## Βήμα 6: Επικόλληση config στα αρχεία

Άνοιξε **index.html** και **admin.html** και βρες αυτό το τμήμα:

```javascript
const firebaseConfig = {
  apiKey:            "YOUR_API_KEY",
  authDomain:        "YOUR_PROJECT.firebaseapp.com",
  projectId:         "YOUR_PROJECT_ID",
  ...
};
```

Αντικατέστησε με το config που αντέγραψες. **Κάνε το ίδιο και στα δύο αρχεία.**

---

## Βήμα 7: Ανέβασμα στο domain σου

Απλά ανέβασε τα δύο HTML αρχεία στον server σου (cPanel, FileZilla, κτλ.):

- `index.html` → `https://yourdomain.com/` (ή `/booking/`)
- `admin.html` → `https://yourdomain.com/admin` (κράτα το URL μυστικό)

**Δεν χρειάζεται PHP, Node.js, ή οτιδήποτε άλλο** — τα αρχεία είναι static HTML.

---

## Πώς λειτουργεί

### Εσύ (admin):
1. Μπαίνεις στο `yourdomain.com/admin.html`
2. Συνδέεσαι με email/κωδικό
3. Προσθέτεις διαθέσιμες ώρες (μεμονωμένες ή επαναλαμβανόμενες)
4. Βλέπεις όλες τις κρατήσεις, ακυρώνεις αν χρειαστεί

### Ασκούμενοι:
1. Μπαίνουν στο `yourdomain.com` (ή όπου έχεις βάλει το index.html)
2. Επιλέγουν ημερομηνία → ώρα → βάζουν όνομα & email
3. Η κράτηση καταχωρείται άμεσα

---

## Κόστος

- **Firebase Firestore**: Δωρεάν έως ~50.000 reads/ημέρα — πολύ παραπάνω από ό,τι χρειάζεται ένα γυμναστήριο
- **Firebase Authentication**: Δωρεάν
- **Hosting**: Ό,τι hosting έχεις ήδη
